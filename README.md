## Run Locally

Clone the project

```bash
  git clone https://github.com/Rasheedatj/Escrow.git
```

Go to the project directory

```bash
  cd Escrow
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
